# CHC 28-Day Timeliness & Delay Analysis (England)

This project analyses NHS England Continuing Healthcare (CHC) and NHS-funded
Nursing Care (FNC) data to understand administrative timeliness and delays.

## Objective
- Analyse the 28-day timeliness standard for CHC referrals
- Identify delays (>28 days) and backlog indicators
- Highlight variation across Sub-ICB locations
- Provide practical insights to support CHC administrative teams

## Data Source
Publicly available NHS England CHC and FNC statistical data
(Q1 2017-18 to Q2 2025-26).

## Tools Used
- Python (Pandas, Matplotlib)
- Google Colab
- Excel / Google Sheets

## Disclaimer
This project uses aggregated, publicly available data only.
No patient-level or confidential information is included.
